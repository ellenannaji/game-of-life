# 🎮 Game of Life

A web-based implementation of Conway's Game of Life built using HTML, CSS, and JavaScript.

## 📌 About The Project

Game of Life is a cellular automaton simulation created by mathematician John Conway. 
This project demonstrates how simple rules can create complex patterns through generations of cells.

The application allows users to interact with a grid, create patterns, and simulate the evolution of cells.

## ✨ Features

✅ Interactive grid-based simulation  
✅ Start / Pause the simulation  
✅ Reset the board  
✅ Generate random patterns  
✅ Real-time cell evolution  
✅ Simple and responsive user interface

## 🎯 Game Rules

The simulation follows Conway's Game of Life rules:

1. A live cell with fewer than two live neighbors dies.
2. A live cell with two or three live neighbors survives.
3. A live cell with more than three live neighbors dies.
4. A dead cell with exactly three live neighbors becomes alive.
